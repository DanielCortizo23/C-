# Gestor de Equipo de Voleibol
Un entrenador quiere llevar un registro de su equipo. Quiere consultar info de sus jugadores, del entrenador y de los partidos

# Objetivo
Sistema que modele 3 tipos de objetos
1 Jugador
2 Entrenador
3 Prtido

# Acciones del usuario por ahorita
Mostrar info de jugadores
Mostrar info del entrenador
Registar partidos, con puntos de los equipos
Simular entrenamientos o mejoras
